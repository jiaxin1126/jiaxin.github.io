Operation In python What does it mean Example
and & Compare position.
If both one, result is one.
Otherwise, zero.		 101 & 011 -> 001
or | Compare position.
If both zero, result is zero.
Otherwise, one.		 101 | 011 -> 111
xor ^ Compare position.
If different, result is one.
Otherwise, zero.		 101 ^ 011 -> 110
not ~ -(x+1) ~101 -> -110
~011 -> -100
left shift << Moved to left. 101 << 2 -> 10100
011 << 2 -> 1100
right shift >> Moved to right. 101 >> 2 -> 001
011 >> 2 -> 0
