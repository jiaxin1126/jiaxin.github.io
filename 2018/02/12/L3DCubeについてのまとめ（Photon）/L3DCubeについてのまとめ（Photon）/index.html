L3DCubeについてのまとめ（Photon）

L3DCube/particle/photon/日本語

このpostはやっぱ日本語と中国語で書いておく。
日本語版と中国語版と別にする。

L3DCube(8x8x8)の基本

L3DCube
一言でいえば、8x8x8個のLED＋Photonというデバイス。
8x8x8LEDは頭がないやつで、ピカピカするか黙るか全部Photonからの命令。
つまり、実際にCubeの本体はPhotonという、下についてるいつも点滅してる、サイズ的にちょっとミニなデバイス。

Photon

Particleという会社の製品。

こいつはすごく面倒くさくて情緒が不安定で、気分（モード）が変わったら表情（ピカピカしてる光の色）も変わる。
まぁこいつとうまく付き合うために一応覚えたほうがいいことを覚えましょう。
デバイスモード

ps.DFUモードが重要。どうやってDFUモードに入るのも重要。リセットの仕方より重要。

pps.Photonはどうやって携帯とか経由でWi-Fiと接続するのかとかのことは上のサイトのDocumentで書いてあるのでここで省略。

Photonと会話する（×）方法

  • Web IDE
  • Desktop IDE

Web IDE

Cubeではなく、単なるPhotonをいじる時に使った。

Desktop IDE

実はAtom.
使い方はWeb IDEと一緒で、ローカルで既存exampleファイルとかがある時にこっちのほうが便利。
ただし、このpostを書く時点で実行を成功したことが一回もない。

Firmware Manager

なので何も言えない。

Particle-cli

情報系だとこれは一番親切でしょう！！！！！
ただしインストールするのは超絶面倒くさい。
少なくとも私がインストースする際にはそうだった。

公式Documentsに従って
npm install -g particle-cli
をしたらエラーだった。
Error info* '! Error loading module 'serialport': Could not locate the bindings file. '
解決法
sudo npm install -g --unsafe-perm node-pre-gyp npm serialport particle-cli

エラーとか出ずに順調にparticle-cliをインストールした後particle setupなどのコマンドも行けた。

本当にうまく行けたかどうかを確認しましょう。
