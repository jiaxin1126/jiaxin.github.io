Triad Types

  • Major
    • root–(four semitones)–third–(three semitones)–fifth
    • (eg. C E G)
  • Minor
    • root–(three semitones)–third–(four semitones)–fifth
    • (eg. C Eb G)
  • Augmented
    • root–(four semitones)–third–(four semitones)–fifth
      *(eg. C E G#)
  • Diminished
    • root–(three semitones)–third–(three semitones)–fifth
    • (eg. C Eb Gb)

Beyond Triad

  • Minor-Major
    • root–(three semitones)–third–(four semitones)–fifth–(four semitones)–seventh
    • (eg. A C E G#)
    • root–minor third–perfect fifth–major seventh
  • Half-diminished
    • root–(three semitones)–third–(three semitones)–fifth–(four semitones)–seventh
    • (eg. B D F A)
    • root–minor third–diminished fifth–flat seventh
  • Augmented-major
    • root–(four semitones)–third–(four semitones)–fifth–(three semitones)–seventh
    • (eg. C E G# B)
    • root–major third–augmented fifth–major seventh
  • Dominant
    • root–(four semitones)–third–(three semitones)–fifth–(three semitones)–seventh
    • (eg. C E G Bb)
    • root–major third–perfect fifth–minor seventh
